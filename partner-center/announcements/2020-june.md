## <a name="now-available-windows-7-year-2-esu-for-cloud-solution-provider-csp"></a><a name="6"></a>Teraz dostępny w systemie Windows 7 rok 2 ESU dla dostawcy rozwiązań w chmurze (CSP)

### <a name="categories"></a>Kategorie

- Data: 2020-06-29
- Typ: oferty/rynki
- Stan: jest teraz dostępny

### <a name="summary"></a>Podsumowanie

Cennik systemu Windows 7, roku 2 Extended Security Updates (EJW) znajduje się teraz w [galerii zasobów](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

### <a name="impacted-audience"></a>Odbiorcy, których dotyczy problem

Wszyscy partnerzy uczestniczący w programie dostawcy rozwiązań w chmurze (CSP)

### <a name="details"></a>Szczegóły

Obsługa systemu Windows 7 zakończyła się 14 stycznia 2020. Chociaż wielu klientów jest dobrze do wdrożenia systemu Windows 10, firma Microsoft rozumie, że wszyscy użytkownicy są w innym punkcie w procesie uaktualniania. Okres obowiązywania ESU systemu Windows 7 Year 1 stycznia 2021. Wiemy, że klienci rozpoczynają planowanie na następny rok.  

W przypadku klientów, którzy oczekują na kontynuowanie korzystania z urządzeń z systemem Windows 7 w 2021, firma Microsoft ustala cennik systemu Windows 7 Year 2 EJW dostępnego dla partnerów dostawcy rozwiązań w chmurze (CSP) teraz, aby pomóc klientom w podejmowaniu decyzji w systemie Windows 10. Aby uzyskać więcej informacji [, odwiedź Galerię zasobów](https://partner.microsoft.com/resources/collection/windows-esu-csp#/) .

### <a name="next-steps"></a>Następne kroki
Udostępnij te informacje odpowiednim kontaktom w organizacji.

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>Aktualizacja funkcji usuwania relacji Centrum partnerskiego

### <a name="categories"></a>Kategorie

- Data: 2020-06-29
- Typ: możliwości
- Stan: jest teraz dostępny
- 
### <a name="summary"></a>Podsumowanie

Od 29 czerwca 2020 dostawcy rozwiązań w chmurze (CSP) i pośrednich odsprzedawcach mogą usuwać relacje podrzędne przy użyciu Centrum partnerskiego i nie będzie już trzeba przesyłać biletu pomocy technicznej Centrum partnerskiego. Pozwoli to zaoszczędzić znaczną ilość czasu partnerom podczas wykonywania tych działań.

### <a name="impacted-audience"></a>Odbiorcy, których dotyczy problem

Dostawcy pośredni dostawcy rozwiązań w chmurze (CSP) i pośrednich odsprzedawcach

### <a name="scenarios-supported"></a>Obsługiwane scenariusze:

- Dostawcy niezależni, którzy mają ustanowioną relację z pośrednim odsprzedawcą mogą usunąć tę relację przy użyciu Centrum partnerskiego.
- Pośrednich odsprzedawcy, którzy mają ustanowioną relację z klientem końcowym, mogą usunąć tę relację przy użyciu Centrum partnerskiego.

Akcje w powyższych scenariuszach muszą być wykonywane przez agenta administratora Centrum partnerskiego.

### <a name="next-steps"></a>Następne kroki:

- Aby uzyskać więcej informacji [, odwiedź Galerię zasobów](https://partner.microsoft.com/resources/collection/partner-center-relationship-removal#) .
- Przejrzyj materiały dotyczące kolekcji z galerii gotowości do działania: Centrum partnerskie — Aktualizacja funkcji usuwania.
- Udostępnij te informacje w organizacji, aby odpowiedni zespół mógł przeglądać i aktualizować procesy.

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>Domyślna częstotliwość rozliczeń zmiana z roczna na miesięczna

### <a name="categories"></a>Kategorie

- Data: 2020-06-17
- Typ: możliwości
- Stan: wkrótce

### <a name="summary"></a>Podsumowanie

Domyślna częstotliwość rozliczeń jest zmieniana z rocznej na miesięczny w środowisku zakupów Centrum partnerskiego.

### <a name="impacted-audience"></a>Odbiorcy, których dotyczy problem

Dostawcy rozwiązań w chmurze (CSP) Direct i dostawcy pośrednim

### <a name="details"></a>Szczegóły

Ustawienie częstotliwości rozliczania określa, jak często jest naliczana opłata za zakupione usługi. Partnerzy widzą to ustawienie w kroku **przeglądowym** podczas zakupu w katalogu Centrum partnerskiego. To ustawienie jest domyślnie **roczne** , ponieważ wprowadzono roczne rozliczenia dla usług opartych na licencji. W oparciu o opinie partnerów zespół Centrum partnerskiego zmienia to ustawienie domyślne od 17 czerwca od **rocznego** do **miesięcznego** , aby lepiej dostosować się do tego, czego partnerzy oczekują. Partnerzy mogą zmienić tę wartość przed przesłaniem transakcji na stronie Przegląd lub po transakcji ze strony zarządzania subskrypcją. Należy pamiętać, że oferty obsługujące tylko roczne rozliczenia będą nadal domyślnie roczne. Ta domyślna zmiana dotyczy tylko środowiska użytkownika portalu Centrum partnerskiego, ale nie ma żadnych zmian w sposobie działania funkcji interfejsów API Centrum partnerskiego.

Firma Microsoft zostaje tę zmianę, co zmniejsza prawdopodobieństwo przypadkowego zakupienia rocznych subskrypcji rozliczanych przez partnera.

- Typ: oferty/rynki

### <a name="impacted-audience"></a>Odbiorcy, których dotyczy problem

Oryginalne oferty F1 zostaną zastąpione w dniu 15 czerwca z nowymi ofertami F1 (poniżej) w przypadku nowych zakupów w ramach subskrypcji.

Partnerzy korzystający z interfejsów API Centrum partnerskiego mogą również korzystać z poniższych informacji.
